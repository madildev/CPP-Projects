To create new folder on github:
echo "# C-" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/Muhammad-Adil-arch/C-.git
git push -u origin master

To push existing folder on github:
git remote add origin https://github.com/Muhammad-Adil-arch/C-.git
git push -u origin master
